# Sistema de Gestión de Inventarios

## Descripción
Este proyecto corresponde a un sistema de gestión de inventarios simple desarrollado en Python, aplicando los fundamentos de la Programación Orientada a Objetos (POO).

El sistema permite administrar productos mediante un menú interactivo en consola.

## Objetivo
Desarrollar un sistema que permita añadir, actualizar, eliminar y buscar productos utilizando clases, métodos y estructuras de datos personalizadas.

## Funcionalidades
- Añadir productos al inventario
- Eliminar productos por ID
- Actualizar cantidad y precio de productos
- Buscar productos por nombre
- Listar todos los productos registrados
- Menú interactivo en consola
